# C14-Engenharia-de-Software
Os grupos devem implementar um pipeline completo de CI/CD utilizando GitHub Actions, contemplando as etapas de teste automatizado, build, deploy e notificação. O foco principal está na qualidade e organização do pipeline e dos testes o software testado pode ser qualquer sistema de escolha do grupo, desde que atenda aos critérios definidos.

Executar projeto
```
uvicorn main:app --reload
```

Executar teste
```
pytest
```
